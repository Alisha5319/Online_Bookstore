# Online_Bookstore
Relational database project for an Online Book Store built using SQL, focusing on database design, query optimization, and sales analysis
This project demonstrates the design and implementation of a relational database for an Online Book Store using SQL.
The database includes tables for books, customers, orders, payments, and inventory management.
Complex SQL queries were written to perform sales analysis, track customer purchase behavior, identify top-selling books, and generate business reports.
The project highlights practical skills in database schema design, joins, subqueries, aggregations, and stored procedures.
